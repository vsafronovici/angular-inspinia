Inspinia UI-Angular
=============================


## Requirements
* Note ! The deployment of dev environment have been tested on OS Centos 7.4

*  Install NodeJS

   - Download NodeJS yum repository:

    ```
      curl -sL https://rpm.nodesource.com/setup_8.x | bash -
    ```

   - YUM Install

    ```
      yum install nodejs-8.6.0       
    ```


*  Install

```
  npm install       
```

## Building NodeJS
```
  yum install nodejs-8.6.0       
```

## Serve NodeJS
```
  npm serve      
```
